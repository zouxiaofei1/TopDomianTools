语言文件制作格式：

ver=x
用于标识语言文件版本，防止出现错误
x值必须和程序中的Update值相同

<Main>
</Main>
<TDR>
</TDR>
用于标识窗口

<Buttons>
"ID" = "按钮名称",左端坐标,上端坐标,宽度,高度,"注释内容"
</Buttons>
注意:	按钮可以没有"注释内容"
	注释中不能有英文双引号
	注释中用\n换行，如果要连续使用两个\n，必须在中间加一个空格
	左端坐标,上端坐标,宽度,高度,可以填-1表示不变

<Checks>
Cx = " 选择框名称",左端坐标,上端坐标,响应宽度
</Checks>
注意:	Checks没有ID，只能用Cx标识，如C1,C2等等
	响应宽度=a  指鼠标在Check右侧a个像素处边框变成蓝色
	Check的"选择框名称"这一字符串前建议加一个空格

<Strings>
"ID" = "字符串内容"
</Strings>
同样用\n换行.


<Frames>
Fx = " 名称 ",左端坐标,上端坐标,宽度,高度
</Frames>

<Edits>
Ex = "内容",左端坐标,上端坐标,宽度,高度
</Edits>
